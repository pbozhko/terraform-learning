# **Terraform practical lessons**

__How to deploy and clear infrastructure:__

1. `export AWS_ACCESS_KEY=...`
2. `export AWS_SECRET_ACCESS_KEY=...`
3. `export AWS_DEFAULT_REGION=...`


4. `cd config/`
5. `terraform init`
6. `terraform plan`
7. `terraform apply`
8. `terraform show`
9. `terraform destroy`
