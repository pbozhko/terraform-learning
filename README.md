# **Terraform practical lessons**

__How to deploy and clear infrastructure:__

0. `export AWS_ACCESS_KEY=...`
1. `export AWS_SECRET_ACCESS_KEY=...`
2. `export AWS_DEFAULT_REGION=...`


1. `cd config/`
2. `terraform init`
3. `terraform plan`
4. `terraform apply`
5. `terraform show`
6. `terraform output`
7. `terraform destroy`
